# Blue Whale

**Navigate complexity. Find common ground. See meaning evolve.**

Blue Whale is a simulation platform that transforms text into symbols, revealing structure, tension, and meaning. It combines a symbolic system (Codex V9) with an evolutionary simulation engine (Telos) to explore how narratives change, compress, and stabilise over time.

## What is Blue Whale?

Blue Whale is an experimental simulation environment.

You provide text — a poem, idea, or statement — and the system converts it into a symbolic form. It then runs a step-based evolution, producing a sequence of symbols that reflects how the input transforms under the system’s rules.

The result is:

- a symbolic sequence  
- a set of metrics  
- a reproducible run  

---

## Core Components

- **Codex V9**  
  A symbolic system used to represent structure and meaning.

- **Telos Engine**  
  A step-based simulation that evolves symbolic sequences over time.

---

## Features

- **Symbolic Transformation**  
  Convert text into structured symbolic sequences.

- **Reproducible Runs**  
  Each simulation uses a seed for consistent results.

- **Metrics Output**  
  Includes measures such as stability and convergence.

- **Adjustable Parameters**  
  Tune system behaviour using available controls.

- **Exportable Data**  
  Simulation output can be exported as JSON.
